ACCESIBILIDAD

Objetivo

Desde HAUDA Producciones pensamos que la accesibilidad es una característica básica de la web que promueve uno de sus principios esenciales: la universalidad. Llegar a cuantos más clientes potenciales, mejor.
Entre las ventajas de una web accesible tenemos:

  • Facilita el acceso de las personas con discapacidad, personas mayores, y en general, de todos los usuarios.
  • Permite el acceso con diferentes navegadores, dispositivos, etc.
  • Conlleva contenidos más claros y estructurados.
  • Mejora la navegación y la experiencia del usuario

Navegación mediante teclado

Para navegar por nuestro sitio web, se pueden utilizar las siguientes teclas de acceso rápido:

  • a: Página Accesibilidad
  • i: Página de Inicio
  • q: Página de Quienes somos
  • s: Página de Servicios
  • m: Página de Contacto
  • Páginas de Trabajos:
    • p: Página de Producción Audiovisual
    • g: Página de Diseño Gráfico
    • f: Página de Fotografía
    • w: Página de Diseño Web
    • c: Página de Comunicación

El uso de las teclas de acceso varía con el navegador:

  • Internet Explorer: ALT atajo
  • Firefox: ALT Mayúsculas atajo (en versiones antiguas ALT atajo)
  • Google Chrome: ALT atajo
  • Opera: MAYÚSCULAS ESC atajo

Compatibilidad

El diseño ha sido realizado para que sea compatible con el mayor número de navegadores posible y en diferentes plataformas.

La mejora de la accesibilidad en los contenidos del portal debe ser un proceso continuo. Para continuar con esta mejora, nos gustaría contar con su opinión. Si tiene alguna duda o sugerencia, puede contactar a través de nuestra página de Contacto.
